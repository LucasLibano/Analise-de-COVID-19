# Dashboard de Análise de COVID-19

## Sobre o projeto

Este projeto tem como objetivo analisar dados globais de COVID-19, passando pelas etapas de coleta, tratamento e visualização dos dados.

---

## Pipeline de Dados

O fluxo do projeto foi estruturado da seguinte forma:

Python (API) → CSV → SQL → Power BI

---

## Coleta de Dados

Os dados foram obtidos a partir de uma API pública de COVID-19 utilizando Python.

---

## Tratamento de Dados

* Conversão de arquivos Excel para CSV
* Organização e padronização dos dados
* Preparação dos dados para análise no Power BI

---

## Dashboard

O dashboard foi desenvolvido no Power BI com foco em análise dos dados:

* KPIs principais:

  * Total de Casos
  * Total de Mortes
  * Total de Recuperados

* Análises:

  * Top 10 países com mais casos
  * Distribuição geográfica (mapa)
  * Comparação de mortes por continente

---

## Insights

* A maior parte dos casos está concentrada em poucos países
* Existe variação significativa na mortalidade entre regiões
* O impacto da pandemia é desigual entre continentes

---

## Objetivo

Desenvolver habilidades em análise de dados, visualização e manipulação de dados com foco em atuação como Analista de Dados Júnior.
